# website-classifier

A Machine Learning project that classifies websites into categories.

## Categories
ecommerce, education, news, technology, health, finance,
tourism, entertainment, sports, government, social_media, legal

## Tech Stack
- Python
- Scikit-learn
- Requests + Playwright
- Pandas, NumPy

## How to Run
```
pip install -r requirements.txt
python main.py
```
